# Beekeeper-Monitor

Offline-fähige PWA für Imker: Hornisse, Varroa, Drohnenbrut, Pollen, Schwarm, Königin.

## Installation
1. Öffne https://norynox.github.io/Beekeeper-Monitor-/
2. Installiere als App.

## Features
- Echtzeit-KI-Erkennung (offline)
- Alarme per Push/E-Mail/SMS (optional)
- Admin-Dashboard
- Mobile-optimiert

Made with Grok!
Yolov10 training of the model is based on https://github.com/computervisioneng/train-yolov10-custom-data-full-guide
